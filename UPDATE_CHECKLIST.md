# 업데이트 및 유지보수 체크리스트

이 문서는 와인 다이어리 프로젝트의 업데이트 또는 배포 작업 시 반드시 수행해야 할 절차를 정의합니다.

## 1. API 연결 상태 점검
N8N 웹훅 서버와의 연결이 정상적인지 확인합니다.
```bash
npm run check-api
```
- **성공 시**: "✨ 모든 API 서버가 정상적으로 응답하고 있습니다." 메시지 확인
- **실패 시**: `.env` 파일의 URL 설정이 올바른지, ngrok 터널이 열려 있는지 확인

## 2. 주요 기능 테스트
- [ ] 홈 화면 로딩 확인
- [ ] **다이어리 탭**: 와인 목록이 정상적으로 불러와지는지 확인 (Failed to fetch 에러 발생 여부)
- [ ] **와인 등록**: 새 와인 등록 및 라벨 분석 기능 작동 확인

## 3. 에러 발생 시 대응
- **Failed to fetch**: 대부분 N8N 서버(ngrok) 주소가 변경되었거나 서버가 꺼져있는 경우입니다.
    1. ngrok 터널 재실행
    2. `.env` 파일의 URL 업데이트
    3. `npm run check-api` 로 재확인
